BACKEND (RESTful API)
=====================
protocol:// domain    :port /resource
http://     localhost :7000 /posts


HTTP METHODS (VERBS):
=====================

CREATE   => POST        (SALVAR)
READ     => GET         (LER / RECUPERAR)
UPDATE   => PUT / PATCH (ATUALIZAR)
DELETE   => DELETE      (REMOVER)
